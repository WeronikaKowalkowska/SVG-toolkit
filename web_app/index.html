Instructions

  • Loading SVG Files: Use the "Choose Files" button to add new designs to your portfolio.
  • Browsing Designs: Click on a design from the list to view it. You can also use the "Previous/Next" buttons.
  • Interacting with the Graphic: Drag the mouse to move the graphic. Use the mouse wheel or the +/- buttons to change the zoom.
  • Rotation: Use the "Rotate" button to rotate the graphic 90 degrees.
  • Reset View: The "Reset View" button restores the default zoom and position settings.
  • Download: Use the "Download SVG" button to save the current design to your computer.
